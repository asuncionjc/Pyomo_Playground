# Pyomo_Playground

Playground used to learn and experiment with algebraic language Pyomo. Do you want to learn something about Pyomo? Then, have a look at 
this repo 🧐.

<img src="https://pyomo.readthedocs.io/en/latest/_images/PyomoNewBlue3.png" height="150" width="600"/>

## Goals

The aim of this repository is to provide some examples of the Pyomo language to practice with them. 
practice with it 📝 . 

Any feedback is welcome, 😄 so please feel free to ask or comment anything you want. If you are shy 😳 and do not want to
write in the repo, I will be happy to answer you by email at asuncionjc@uma.es 👩🏻‍💻 .
